Framework JavaScript

INICIO

Framework de aplicaciones web Node.js. Características: Código abierto lanzado en 2010 Desarrollado en JavaScript Esta bajo la fundación de OpenJs Ventajas: Configuración rápida para aplicaciones web básicas Liviano Paradigma E/S sin bloqueo Desventajas: Mayor cantidad de lineas para una misma aplicación en comparación a otros frameworks Requiere mayor tiempo para construir modelos de aplicaciones en comparación a otros frameworks. Tipo de licencia: MIT Análisis de curvatura de aprendizaje: Debido a su estructura ligera, su curva de aprendizaje se considera corta.

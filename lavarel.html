Framework PHP

INICIO

Framework web dedicado a PHP. Características: Lanzamiento: 2011 soporta la ultima version de PHP 8, incluida la version 5 y 7 Su escritura de codigo se denomina “espagueti” Utiliza la arquitectura Modelo vista controlador Influenciada por Ruby on Rails Ventajas: Puede utilizar servidores web Nginx, IIS o Apache Se destaca por sus sintaxis en su código escrito en PHP Utiliza scripting interfaz interactivaDesventajas: No soporta versiones inferiores a PHP 5.4 Tipo de licencia: MIT Análisis de curvatura de aprendizaje: Debido a su extensa aplicación, su curva de aprendizaje se considera lenta.
